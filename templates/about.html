About Latitude

This website will show visualizatons that were created in previous assignments.

The primary objective is to demonstrate how to use several web development concepts to display information.

# Site Objectives
1 Has the name of the site on the left of the nav which allows users to return to the landing page from any page.
2 Contains a dropdown menu on the right of the navbar named "Plots" that provides a link to each individual visualization page.
3 At the top of every page, have a navigation menu.
4 Provides two more text links on the right: "Comparisons," which links to the comparisons page, and "Data," which links to the data page.
5 Is responsive (using media queries). The nav must have similar behavior as the screenshots "Navigation Menu" section (notice the background color change).
